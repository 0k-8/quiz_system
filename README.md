# 🎓 Quiz System (Навчальний проєкт)

[![Java Version](https://img.shields.io/badge/Java-22-orange.svg)](https://www.oracle.com/java/)
[![Build Status](https://github.com/0k-8/quiz_system/actions/workflows/build.yml/badge.svg)](https://github.com/0k-8/quiz_system/actions)

Цей проєкт розробляється в рамках **практичного вивчення** мови Java та принципів побудови архітектури корпоративних додатків. На даному етапі фокус зосереджений на роботі з базами даних та створенні надійної інфраструктури.

---

## 🏗 Поточний стан (Stage: Infrastructure & DAO)

Проєкт знаходиться в активній фазі розробки. Реалізовано базовий шар роботи з даними:
- **Infrastructure**: Пул з'єднань (Connection Pool), налаштування середовища.
- **DAO Pattern**: CRUD операції для всіх основних сутностей.
- **Database Migrations**: Повна історія змін схеми БД через Flyway.
- **CI/CD**: Налаштована автоматична збірка інсталяторів через GitHub Actions.

## 🔧 Як запустити локально

1. Клонуйте репозиторій.
2. Впевніться, що у вас встановлена **Java 22**.
3. Зберіть проєкт: `mvn clean package`.
4. Запустіть: `java -jar target/quiz-system-1.0-SNAPSHOT.jar`.
