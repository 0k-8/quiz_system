# 🎓 Quiz System

[![Java Version](https://img.shields.io/badge/Java-22-orange.svg)](https://www.oracle.com/java/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Build Status](https://github.com/0k-8/quiz_system/actions/workflows/build.yml/badge.svg)](https://github.com/0k-8/quiz_system/actions)

Професійна система для проведення командних вікторин (квізів). Проєкт реалізований на Java з використанням сучасних архітектурних підходів та надійного шару обробки даних.

## ✨ Ключові можливості

- 🚀 **Висока швидкість**: Власний Connection Pool для миттєвої роботи з БД.
- 🛡️ **Надійність**: Повна підтримка транзакцій та цілісності даних (DAO Pattern).
- 🔄 **Автоматизація**: Міграції бази даних через Flyway (v1..v8 вже інтегровані).
- 🎨 **Сучасний GUI**: Інтуїтивний інтерфейс на JavaFX.

---

## 🛠 Технологічний стек

- **Мова:** Java 22
- **Графіка:** JavaFX
- **База даних:** H2 (Embedded)
- **Міграції:** Flyway
- **Збірка:** Maven + GitHub Actions (CI/CD)

---

## 🏗 Архітектура проєкту

Система побудована на багатошаровій архітектурі:
1. **View**: JavaFX компоненти для взаємодії з користувачем.
2. **ViewModel**: Логіка обробки даних для UI (MVVM).
3. **Service**: Бізнес-логіка програми.
4. **DAO (Data Access Object)**: Рівень доступу до даних.
5. **Infrastructure**: Налаштування з'єднань, пул потоків та безпека.

## 🔧 Встановлення для розробників

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/0k-8/quiz_system.git
   ```
2. Зберіть проєкт за допомогою Maven:
   ```bash
   mvn clean package
   ```
3. Запустіть програму:
   ```bash
   java -jar target/quiz-system-1.0-SNAPSHOT.jar
   ```
